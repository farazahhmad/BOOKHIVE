About BookHive

BookHive is a simple and efficient library management system created for students and librarians. It allows users to search books, check availability, borrow books, return them, and manage the library database ‚Äî all in one clean, easy-to-use interface.

How It Works

  • Students can search for books in real time.
  • Borrow books instantly if available.
  • Return books from their dashboard.
  • Librarians can add, update, or delete books.
  • All data is stored locally using JSON files.
